## Configuration File Example

Default location: ~/.crypto-trader.yaml

```
# KuCoin API key
kucoin.api.key: xxx

# KuCoin API secret
kucoin.api.secret: xxx

# QuadrigaCX
quadriga.api.client-id: xxx
quadriga.api.key: xxx
quadriga.api.secret: xxx

# Kraken
kraken.api.key: xxx
kraken.api.secret: xxx
```
